# Alireza Olama - Personal Academic Website

Personal research website for Alireza Olama, Postdoctoral Researcher at Åbo Akademi University specializing in **Machine Learning Systems for Energy-Efficient Large-Scale AI**.

## Research Focus

- Energy-efficient distributed training of foundation models
- Sparsity-aware optimization and structured pruning
- Hardware–software co-design for inference
- Communication-efficient parallelism
- Federated and decentralized learning

## Open-Source Software

- **PruneX**: Hierarchical pruning-aware distributed training system
- **PsFIT**: GPU-accelerated Parallel Sparse Fitting Toolbox
- **SCOT**: Sparse Convex Optimization Toolkit

## Setup

1. **Add your photo**: Add `photo.jpg` (recommended: 400x400px+, square aspect ratio)

2. **Update Google Scholar link**: Replace `YOUR_ID` in `index.html` with your actual Google Scholar ID
